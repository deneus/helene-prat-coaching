# Plan : Site helene-coaching

## Contexte
Créer un site vitrine one-page pour Hélène Prat, coach certifiée ORSC et formatrice PCM, basée à Hilversum (Pays-Bas). Le site présentera son profil et son programme "Rooted Together" pour les couples en relocation. Contenu source dans `docx/txt/`.

## Inspirations design (4 sites analysés)

| Site | Éléments retenus |
|------|-----------------|
| **Cortney McDermott** | Layout épuré, taupe (#b1a69d), navigation en majuscules espacées, footer sombre, interligne 1.65 |
| **Kamala (WordPress)** | Palette douce naturelle (beige, vert sage, rouille), typographies premium serif/sans-serif, espacement fluide généreux |
| **ChicSavvy** | Boutons arrondis (pill shape), CTA multiples, section "Meet your coach" avec photo+bio, tons corail/crème (#fff5ec) |
| **Mental Care** | Boutons pill (#423466), police Mulish, hero image-driven, 3 templates de sections, ambiance confiance/professionnelle |

### Synthèse design retenu
- **Ambiance** : Serein, naturel, professionnel et chaleureux (mix Kamala + Cortney)
- **Boutons** : Pill shape (border-radius: 9999px) comme ChicSavvy et Mental Care
- **Layout** : Alternance sections claires/colorées, beaucoup de respiration (Kamala)
- **Navigation** : Majuscules espacées, sobre (Cortney McDermott)
- **Section bio** : Photo + texte côte à côte (ChicSavvy "Meet your coach")
- **CTA** : Multiples points d'appel à l'action tout au long de la page

## Choix techniques
- **HTML/CSS/JS statique** - pas de framework, pas de build
- **One-page** avec navigation sticky par sections
- **Bilingue EN/FR** via toggle JS (spans `lang-en` / `lang-fr` dans le même HTML)

## Système bilingue

```html
<span class="lang-en">English text</span>
<span class="lang-fr">Texte français</span>
```

- CSS : `.lang-fr { display: none; }` par défaut (EN actif)
- `body.fr .lang-en { display: none; } body.fr .lang-fr { display: inline; }`
- JS toggle ajoute/retire classe `fr` sur body
- Préférence sauvée en localStorage

## Étapes d'implémentation

1. **`css/style.css`** - Variables CSS, reset, layout, composants, animations, responsive
2. **`js/main.js`** - Toggle langue, smooth scroll, nav sticky, menu mobile, fade-in scroll
3. **`index.html`** - Structure HTML complète avec tout le contenu EN + FR traduit
4. **`images/`** - Créer dossier + placeholder SVG pour portrait
